## 🎯 Repository Purpose
This repository is established with two primary goals:
1.  **Development Practice:** Enhancing UiPath technical skills through hands-on implementation of complex API integrations and automation logic.
2.  **Jira Optimization:** Developing functional tools to automate repetitive tasks within Jira Software, improving project management efficiency.

## 📁 Project Portfolio
Current automated solutions in this repository:

* **🏷️ My_jira_autoLabelinAssignin**
    * Automates the process of issue labeling and member assignment based on specific business rules.
* **📊 My_jira_weeklyReport**
    * Generates weekly status reports by extracting and processing task data directly from Jira.

## 🏗️ Directory Structure
Each project is isolated in its own folder:

* `📂 /My_jira_autoLabelinAssignin/`: Workflow files for labeling and assignment logic.
* `📂 /My_jira_weeklyReport/`: Project files for reporting and data extraction.
